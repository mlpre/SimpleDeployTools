## 简单部署工具

[English Document](README.md)

* ### 一个简单的部署工具

    * 基于JavaFx
    * 支持英文和中文
    * 支持连接SSH服务器
    * 支持一键部署shell脚本

* ### 如何使用:

    * 下载jar，执行java -jar SimpleDeployTools-1.0.jar

* ### 注意事项:

    * 默认端口22，如果需要使用其它端口，IP输入框输入IP:端口

* ### 主要依赖

    * [jediterm](https://github.com/JetBrains/jediterm)

    * [jsch](http://www.jcraft.com/jsch/)
